# URS - EDMS

| URS_ID | User Requirement |
|---|---|
| URS-001 | System shall enforce role-based access for Author, Reviewer, Approver, Admin. |
| URS-002 | System shall enforce lifecycle Draft → Review → Approved → Archived. |
| URS-003 | System shall bind electronic signatures to a specific document version with meaning. |
| URS-004 | System shall keep immutable, append-only, UTC audit trails for all critical actions. |
| URS-005 | System shall lock approved records from further modification. |
| URS-006 | System shall provide secure authentication with password hashing and JWT sessions. |
| URS-007 | System shall enforce ALCOA+ data integrity (attributable, legible, contemporaneous, original, accurate). |
